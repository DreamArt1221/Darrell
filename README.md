# 图片压缩工具

一个简单易用的在线图片压缩工具，采用苹果风格设计。

## 功能特点

- 支持 PNG 和 JPG 格式图片
- 拖放上传或点击上传
- 实时预览压缩效果
- 显示压缩前后文件大小
- 可调节压缩质量
- 支持下载压缩后的图片

## 使用方法

1. 直接拖放图片到上传区域，或点击上传区域选择图片
2. 使用质量滑块调整压缩程度
3. 实时预览压缩效果
4. 点击下载按钮保存压缩后的图片

## 技术栈

- HTML5
- CSS3
- JavaScript
- browser-image-compression 库

## 本地运行

1. 克隆仓库
```bash
git clone https://github.com/DreamArt1221/Darrell.git
```

2. 使用本地服务器打开项目（例如 VS Code 的 Live Server 插件）
